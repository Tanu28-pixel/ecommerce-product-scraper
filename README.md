🛒 E-commerce Product Scraper

📖 Description

This project is a Python-based web scraper that extracts product details such as name, price, ratings, and product links from e-commerce websites like Amazon and Flipkart.

The application allows users to enter a search query (e.g., *laptop*, *mobile*), fetches relevant product listings, and stores the data in a structured CSV file for further analysis.

---

🚀 Features

* 🔍 Search products using user input
* 📦 Extract product details:

  * Product Name
  * Price
  * Rating
  * Product Link
* 📄 Save data into CSV file
* ⚠️ Basic exception handling for stability

---

🛠️ Tech Stack

* Python
* Requests
* BeautifulSoup (bs4)
* CSV / File Handling
